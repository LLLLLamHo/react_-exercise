## 自定义组件

一个比较简单的案例,是一个简单的父组件嵌套子组件。

复杂一点的案例是通过子组件调用父组件传入的,告诉父组件刷新子组件。
自定义组件是一个两个单选框的案例,通过父组件更新状态后改变子组件,从新渲染新的组件。
